acessar pasta com o back, criar a env com o comand:
 	python - venv env 
depois ativar a env
	env\scripts\activate
	pip install -r requirements.txt -> instala tudo como por exemplo o django, pandas e etc 
em seguida acessar a pasta com o comando
	cd cidade_inteligente
em seguida devemos start o back 
	python manage.py runserver

Depois devemos start o front
Acesse a pasta do front
de
	npm install
	npm install axios
	npm install react-leaflet leaflet   
	
Após isso podemos start o front com o comando
	npm start

OBS: CASO DE ALGUM PROBLEMA NO FRONT RELACIONADO AS INSTALAÇÕES E RECOMENDADO APAGAR O NODE_MODULES E O PACKAGE-LOCK.JSON E REINSTALAR TUDO
